
# Docker Hub Secret Setup Instructions

1. Go to your GitHub repository: https://github.com/tmachinya/hedge-funds-data
2. Click on 'Settings' > 'Secrets and variables' > 'Actions'
3. Add the following secrets:

- Name: DOCKER_HUB_USERNAME
  Value: tmachinya

- Name: DOCKER_HUB_TOKEN
  Value: (Your Docker Hub access token)

Generate your token here: https://hub.docker.com/settings/security
